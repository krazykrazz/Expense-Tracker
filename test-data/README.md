# Test Data

This directory contains test data files used for development and testing purposes.

## Contents

- Sample data files for testing
- Test database files (gitignored)

## Valid Expense Categories

The system supports the following expense categories:

**Essential Living:**
- Housing
- Utilities
- Groceries
- Insurance

**Transportation:**
- Gas
- Vehicle Maintenance

**Food & Dining:**
- Dining Out

**Entertainment & Lifestyle:**
- Entertainment
- Subscriptions
- Recreation Activities

**Personal:**
- Personal Care
- Clothing
- Gifts

**Family & Pets:**
- Pet Care

**Tax-Deductible:**
- Tax - Medical
- Tax - Donation

**Other:**
- Other

## Usage

These files are for development use only and should not be committed to version control.

## Note

All data files in this directory are ignored by git to prevent accidental commits of test data.
